# Professor Assistant – Exam Generator (Python)

A simple and user-friendly Python program that helps professors generate exams from a question bank.
The program loads Q/A pairs from a text file and randomly selects questions to create an exam and answer key.

## Features
- Loads a text-based question bank
- Randomly selects question–answer pairs using `random.randint()`
- Creates an exam file and answer key
- Simple terminal interface
